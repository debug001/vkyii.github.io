# 释放鼠标

默认是`Ctrl+Atl`释放鼠标,并且在__PD偏好设置 -> 快捷键 -> 应用程序快捷键__中可以看到,但是是不能更改的.

解决方法是: 进入__系统偏好设置 -> 键盘 -> 应用程序快捷键__, 增加一个Parallels Desktop的快捷键,并且名字与PD里面的那个一致(中英文版本不同).

修改完以后就直接生效了,以后在PD里面用像VMware这样的虚拟机就不会因为鼠标的问题而卡死了.
